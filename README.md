## 本地运行&开发

克隆项目

```bash
  git clone https://github.com/juxiu/g-crx-vue.git
```

进入目录

```bash
  cd g-crx-vue
```

安装依赖

```bash
  npm install
```

运行

```bash
  npm run build:watch
```

chrome 扩展管理中加载已解压的扩展程序,选择 dist 目录
